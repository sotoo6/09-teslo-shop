## Teslo Shop

Tienda online de Teslo

# Levantar proyecto en Dev:

1. Clonar repositorio
2. Clonar repositorio `10-nest-teslo-shop-backend`
3. En `10-nest-teslo-shop-backend` levantar db con `docker-compose up -d`
4. En `10-nest-teslo-shop-backend` levantar proyecto con `npm run start:dev`
5. En `09-teslo-shop` instalar dependencias
6. Levantar proyecto con `ng serve`
